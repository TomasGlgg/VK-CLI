# VK-CLI
VKontakte CLI version

## Команды главной консоли:
**Показать справку:**

`help`

**Добавить новый токен в список:**

`add <токен>`

**Список добавленых токенов:**

`list`

**Удалить токен из списка:**

`delete <id токена>`

**Авторизоваться с помощью токена:**

`auth <id токена>`

**Выйти:**

`exit`

### Пример:
```
Список токенов загружен
(VK-CLI)list
0 7dd020931e...
(VK-CLI)auth 0

      Телефон: +7 *** *** ** 37
       Страна: Россия
       Статус: 
(Эмиль Иванов)>
```
 

 ## Команды профиля:
 
 **Показать справку:**
 
 `help`
 
 **Вывести диалоги:**
 
 `dialogs [количество]`

 **Вывести непрочитанные диалоги:**

 `unread [количество]`
 
 **Выбрать диалог:**
 
 `select <id чата>`

 **Показывать сообщения в реальном времени:**

 `online [аргументы]`

 **Выйти из профиля:**

 `exit`


 ## Команды приватного диалога:
 
 **Прочитать сообщения:**
 
 `read [кол-во]`

 **Выйти из приватного диалога:**
 
 `exit`

 **Написать сообщение:**

 `write [сообщение]`

 **Показывать сообщения собеседника в реальном времени:**

 `online [аргументы]`


 ## Команды чата:

 **Прочитать сообщения:**

 `read [кол-во]`

 **Написать сообщение:**

 `write [сообщение]`

 **Выйти из чата:**

 `exit`
