# VK-CLI
VKontakte CLI version

## main cmd commands:
**Показать справку:**

`help`

**Добавить новый токен в список:**

`add <token>`

**Список добавленых токенов:**

`list`

**Удалить токен из списка:**

`delete <token id>`

**Авторизоваться с помощью токена:**

`auth <token id>`

### Пример:
```
Список токенов загружен
(Cmd) list
0 7dd020931e...
(Cmd) auth 0

        Phone: +7 *** *** ** 37, Country: Россия
        
        Status:  
        
(Эмиль Иванов)>
```
